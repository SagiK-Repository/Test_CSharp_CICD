# 1. 목표 선정  

- 다음 조건을 만족한다.  

1. Visual Studio를 통해 C# sln 파일을 만든다. 
2. C# Test 프로젝트를 구성한다.
3. 특정 브랜치에 Commit 될 때 CICD가 동작한다.
4. CICD를 통해 자동 빌드 및 Test를 진행한다.
5. 문제가 없으면 자동 배포된다.
